# `eslint-config-simenb-flow`
> AVA ESLint config

[![NPM Version][npm-image]][npm-url]

Flow type config.

## Usage

```sh
$ npm install --save-dev eslint-config-simenb-flow
```

`.eslintrc`:

```json
{
  "extends": "simenb-flow"
}
```
