# `eslint-config-simenb-base`

[![NPM Version][npm-image]][npm-url]

Base ESLint config.

## Usage

```sh
$ npm install --save-dev eslint-config-simenb-jsdoc eslint-plugin-jsdoc
```

`.eslintrc`:

```json
{
  "extends": "simenb-jsdoc"
}
```
