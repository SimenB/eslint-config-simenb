# `eslint-config-simenb-base`
> Base ESLint config

[![NPM Version][npm-image]][npm-url]

Base ESLint config.

## Usage

```sh
$ npm install --save-dev eslint-config-simenb-base
```

`.eslintrc`:

```json
{
  "extends": "simenb-base"
}
```
