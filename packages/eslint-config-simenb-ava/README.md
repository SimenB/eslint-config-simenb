# `eslint-config-simenb-ava`
> AVA ESLint config

[![NPM Version][npm-image]][npm-url]

AVA config.

## Usage

```sh
$ npm install --save-dev eslint-config-simenb-ava eslint-plugin-ava babel-eslint
```

`.eslintrc`:

```json
{
  "extends": "simenb-ava"
}
```
