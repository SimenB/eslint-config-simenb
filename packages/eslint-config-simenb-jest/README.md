# `eslint-config-simenb-jest`
> Jest ESLint config

[![NPM Version][npm-image]][npm-url]

Jest config.

## Usage

```sh
$ npm install --save-dev eslint-config-simenb-jest
```

`.eslintrc`:

```json
{
  "extends": "simenb-jest"
}
```
