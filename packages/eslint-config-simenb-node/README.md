# `eslint-config-simenb-node`
> Node ESLint config

[![NPM Version][npm-image]][npm-url]

Node config.

## Usage

```sh
$ npm install --save-dev eslint-config-simenb-node
```

`.eslintrc`:

```json
{
  "extends": "simenb-node"
}
```
